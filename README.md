# fasta-basic-stats

> This tool is open source. Feel free to try, modify, or use it in your projects. And please give me feedbacks

A lightweight command-line Python tool to analyze FASTA files and generate comprehensive reports, including sequence counts, GC content, nucleotide composition, and sequence length statistics.

---

## Features

- Count total sequences in a FASTA file  
- Check for unique headers and identify duplicates  
- Compute sequence length statistics (min, max, mean, median)  
- Calculate overall GC content  
- Calculate GC content per sequence  
- Compute nucleotide composition (counts and proportions)  
- Generate a full report saved as a TXT file
